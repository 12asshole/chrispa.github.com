Getting Started with Trillo

This guide has been written to help new customers get started using Trillo. It will walk you through the basic steps we reccomend you take to get started, with the specific goal of testing out how Trillo works to get a sense of if it will solve your current registration and class management problems.

As always we are only an email or phone call away. Please contact us at help@trilloclass.com or call 802-318-4714 if you run into problems. Thank you for trying Trillo!

Just click - you ain't gonna break it!

Trillo comes with a little bit of 'dummy data' to help you see how things work. There is a class already set up, with two students enrolled. There is also one Registration form for you to review. So our #1 piece of advice is to start clicking & editing things. You'll quickly get a feel for how it all works together.

1) Setting Up Your Organization

The first screen you see after signing up for Trillo is the Organization Settings page. To return to this page in the future just click the 'Settings' link in the upper right. Let's set a few things up:

  1. Email Address: This is email where your customers can contact you. It will be the 'Reply-To' email for any emails Trillo sends on your behalf (like registration confirmation emails.)
  2. Waivers: These are two waivers that will show up on EVERY registration form you create. You will also have the ability to create Registration Form Specific waivers when you create the forms. For now let's just put something in both of these fields. Then when you view the registration form you create later, you can see where the language is inserted.
  3. Collecting Payment: We allow you to connect with a service called Stripe in order to collect credit cards. For now let's NOT connect the Stripe account (even if you have one.) It's easier to see how everything works first and then come back and add payment collection.
  4. Users: These are users who can access your Trillo account. Users who are not admins can do everything except access this Settings page.
  5. Sessions: Trillo requires you to organize classes by what we call Sessions. You can have as many as you would like, and you can create different registration forms for different sessions. Create a few here now.
  6. Locations: If you have multiple locations you can enter those here. Locations is used simply as a filter. So for example you can filter/email/export all the classes at a specific location.
  7. Instructors: Just like Locations, Instructors is used to filter & organize classes. Each class can only have one instructor, so if you have classes where there are two you'll need to create a separate instructor. For example: Julie Smith/Wendy Davis

That's it for Settings! Click 'Update Settings' at the bottom of the page.

2) Setting Up Classes

Next, click on the Classes tab on the top navigation bar. You'll see there is an example class already created called 'Drawing With Chalk'. Next click the button on the right side 'Add Class'.

Note that a Class in Trillo is a distinct instance of that class. So if in the Fall session you have three different 'Tumbling For Tots' (one on Monday, one Wed, one Fri) then you should create three different classes in Trillo. (We'll show you how to duplicate classes later!)

  1. Name: The class name!
  2. Active?: If a class is marked Active then it WILL show up on Registration forms. If it is NOT active it will not. A handy way to add future classes without letting people register for them yet, or pull a class from the Registration form.
  3. Description: This field currently does NOT show up in any public facing page. Use if you want, but not required.
  4. Days: Choose the days this class will be offered.
  5. Start/End Dates: When the class starts and ends. Note that once the class is over it will automatically show up in Past Classes. You can click on 'View Past Classes' on the Classes home screen.
  6. Time: When the class is held. Note that this field can be any text - Trillo doesn't do any time calculations. So you can enter things like 'Call To Schedule'
  7. Cost: Total cost of the class. You can enter zero if the class is free.
  8. Kid/Adult/Teen: Who is the class for? This is used for filtering when creating Registration forms. You can select multiple categories.
  9. Max Enrollment: Trillo tracks the number of enrolled students. When a class reaches this limit new registrations aren't allowed. No more wildly oversubscribed classes!! The Details: The class is still seen on the Registration form, but the button to select it disappears. A note next to the class will say 'Class Full - Please email for waitlist.' At this time Trillo doesn't automatically handle waitlist (but it's something in the pipeline.)
  10. Session/Instructor/Location: Specific details to help organize classes.

That's it! Click 'Create Course' to create the class.

3) Setting Up A Registration Form

Next we'll go through the options available to set up a new registration form that your customers can use to register for your classes. You can have multiple Registration forms in Trillo and each will have a unique URL that you can point to on your website.

Why would I want multiple registration forms?

  • You have some classes where you want to capture a LOT of emergency contact information, but others where you don't want to capture any.
  • You might have a separate form for Events or Birthday parties where no emergency contact info is collected.
  • You want to collect just a registration fee for some classes, but the full class cost for others.
  • You have specific instructions or waivers JUST for a subset of your classes.

Here are the form fields and how they affect the Registration form:

  1. Name: The name of the form! This does show up at the top of the public form.
  2. What classes?: Forms are filtered based on Sessions and on Class Type (below)
  3. Class Type?: Are there only certain types of classes we should include? Check all if you want all classes to be included.
  4. What kind of Emergency Contact Info Should we collect?: Comprehensive is the most, Basic is more streamlined, and none skips that section altogether. Note that for Adults we ONLY collect Basic information, even if Comprehensive is checked off. In other words - Comprehensive vs Basic only applies to children.
  5. Print out?: Do you want customers to have an option at the end of the form to print it out and mail it in with a check?
  6. Specific Instructions: These Instructions show up at the top of the form on every page of the form.
  7. Specific Legal Waiver: This waiver shows up on Page 2 AND Page 3. The best advice regarding the waiver info is to try something, then view the form to see where it is located.

That's it! Clicking Create will take you back to all the Registration forms. Clicking the 'View' button to the right of the form will open the form in another tab in your browser. Clicking 'Copy' will copy the URL of the form to your computer clipboard. This is the URL you should point your 'Register Now' button to on your website.
