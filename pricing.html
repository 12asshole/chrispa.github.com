Frequently Asked Questions


Do I have to use Stripe to accept credit cards?

Stripe is the only provider that we have deep integration with. But you don't have to use Stripe - you can turn off payment collection on a per-registration form basis.


What makes Trillo different from other providers like Jackrabbit?

Trillo is built with the latest web technologies to provide YOUR customers with the best registration experience. Trillo is a much simpler program - we don't believe more features means something is better. We don't sell you training because you aren't going to need it!

 


Do I have to install anything?

Nope - Trillo is a web based application, so you access the software through any modern web browser. All you need is an Internet connection. Nothing to install, nothing to patch or maintain, and nothing to back-up. We take care of all of that for you.


Can we import historical information?

Yes - that is handled on a case-by-case basis and we want to make sure Trillo is right for you first.
